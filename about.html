О проекте

Проект "Dark Fantasy Pokémon Cards" представляет собой веб-сайт, который переосмысливает традиционные карты покемонов в стиле темного фэнтези. Наш сайт предлагает уникальный взгляд на мир покемонов, погружая их в мрачную, таинственную атмосферу.

Сайт разработан с использованием современных веб-технологий:

  • HTML5 для структуры страниц
  • CSS3 для стилизации в темном фэнтезийном стиле
  • JavaScript для динамического взаимодействия с API

Все данные о картах покемонов получены с использованием открытого API TCGdex, которое предоставляет обширную информацию о картах покемонов из различных наборов.

Информация об API

TCGdex API - это открытый интерфейс программирования приложений, который предоставляет доступ к обширной базе данных карт покемонов на нескольких языках. API не требует аутентификации и предлагает простой способ получения информации о картах.

Основные эндпоинты API, используемые в проекте:

GET https://www.tcgdex.net/api/v2/cards

Получение списка всех доступных карт

GET https://www.tcgdex.net/api/v2/cards/{id}

Получение детальной информации о конкретной карте по её ID

GET https://www.tcgdex.net/api/v2/sets

Получение списка всех доступных наборов карт

Более подробную информацию об API можно найти на официальном сайте TCGdex.

Проект разработан в образовательных целях для демонстрации работы с API и создания стилизованных веб-интерфейсов.

Используемый API

Проект использует PokeAPI - бесплатный и открытый API для получения данных о покемонах.

Основные эндпоинты API:

  • /pokemon - получение списка всех покемонов
  • /pokemon/{id} - получение детальной информации о покемоне по ID
  • /pokemon-species/{id} - получение информации о виде покемона

Все данные о покемонах получены с использованием открытого API PokeAPI, которое предоставляет обширную информацию о покемонах, их характеристиках, способностях и изображениях.
