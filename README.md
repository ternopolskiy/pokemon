# Dark Fantasy Pokémon Cards

## Описание проекта

Проект "Dark Fantasy Pokémon Cards" представляет собой веб-сайт, который переосмысливает традиционных покемонов в стиле темного фэнтези. Сайт предлагает уникальный взгляд на мир покемонов, погружая их в мрачную, таинственную атмосферу.

Проект разработан с использованием HTML, CSS и JavaScript, и взаимодействует с открытым API PokeAPI для получения данных о покемонах.

## Функциональность сайта

### Главная страница (index.html)
- Приветственный баннер с описанием проекта
- Секция с популярными покемонами, случайно выбранными из API
- Навигация по сайту

### Страница карт (cards.html)
- Отображение коллекции покемонов
- Функция поиска покемонов по названию с адаптивным обновлением результатов
  - Автоматический возврат к полному каталогу при очистке поискового запроса
- Пагинация для навигации по большому количеству покемонов
  - Возможность перемещаться между страницами с помощью кнопок "Предыдущая" и "Следующая"
  - Автоматическое отключение кнопок пагинации при достижении первой или последней страницы

### Детальная страница карты (card-detail.html)
- Подробная информация о выбранном покемоне
- Увеличенное изображение покемона
- Характеристики покемона (тип, способности, статистика и т.д.)

### Страница о проекте (about.html)
- Информация о проекте и его целях
- Описание используемого API и его эндпоинтов
- Технические детали реализации

## Используемое API

В проекте используется PokeAPI - открытый интерфейс программирования приложений, который предоставляет доступ к обширной базе данных покемонов. API не требует аутентификации и предлагает простой способ получения информации о покемонах, их характеристиках, способностях и изображениях.

### Основные эндпоинты API

1. `GET https://pokeapi.co/api/v2/pokemon`
   - Получение списка всех доступных покемонов
   - Используется для отображения коллекции покемонов на странице cards.html
   - Также используется для случайного выбора популярных покемонов на главной странице

2. `GET https://pokeapi.co/api/v2/pokemon/{id}`
   - Получение детальной информации о конкретном покемоне по его ID
   - Используется для отображения подробной информации на странице card-detail.html
   - Также используется для получения полной информации о покемонах в коллекции

3. `GET https://pokeapi.co/api/v2/pokemon-species/{id}`
   - Получение информации о виде покемона
   - Может использоваться для получения дополнительной информации о покемоне

## Технические особенности

### Фронтенд
- **HTML5** для структуры страниц
- **CSS3** для стилизации в темном фэнтезийном стиле
  - Использование современных CSS-свойств (flexbox, grid, transitions)
  - Адаптивный дизайн для различных устройств
  - Кастомные шрифты для усиления атмосферы темного фэнтези
- **JavaScript** для динамического взаимодействия с API
  - Асинхронные запросы к API с использованием Fetch API
  - Динамическое создание элементов DOM
  - Обработка событий пользовательского интерфейса
  - Реализация пагинации для навигации по большому количеству данных
  - Адаптивный поиск с мгновенным обновлением результатов

### Работа с API
- Использование асинхронных функций для запросов к API
- Обработка ошибок при неудачных запросах
- Кэширование данных для оптимизации производительности
- Эффективная загрузка данных с использованием пагинации

## Запуск проекта

Для запуска проекта достаточно открыть файл `index.html` в любом современном веб-браузере. Проект не требует установки дополнительных зависимостей или настройки сервера, так как все взаимодействие с API происходит на стороне клиента.

Альтернативно, вы можете использовать простой HTTP-сервер для запуска проекта:

```bash
python -m http.server 8000
```

После этого проект будет доступен по адресу `http://localhost:8000`.
